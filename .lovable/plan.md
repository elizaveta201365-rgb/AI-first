## Hero-секция для лендинга-портфолио

Создам Hero на главной странице (`src/routes/index.tsx`) в стиле референса: мятно-зелёная палитра, серифный display-заголовок, «сюрреалистичная» композиция с плавным переходом в следующий блок через SVG-волну.

### Визуальный стиль (из референса)
- Палитра: мятно-зелёный фон `#c9dfd0` → `#a8c9b4`, кремовый `#f5f1e8`, тёмно-зелёный акцент `#2d5a4a`, коралловый/оранжевый акцент `#e8927c`, графит `#1a2420`.
- Типографика: display-serif для H1 (Instrument Serif / Fraunces) + гротеск для body (Inter/Manrope). H1 — крупный, с широкими межстрочными.
- Композиция: слева — заголовок/подзаголовок/CTA (как «Wilner / Boe / Dene» в референсе), справа — визуальный «объект» (мок дашборда) с органическими формами и «туманом».
- Плавный переход: внизу секции — SVG-волна кремово-мятного цвета, перетекающая в следующий блок (как в референсе после машины/камней).

### Что меняю
- **`src/routes/index.tsx`** — заменяю плейсхолдер на `<Hero />` + пустой «следующий блок» как приёмник волны (только визуальная стыковка, контента не добавляю).
- **`src/components/Hero.tsx`** (новый) — сама секция.
- **`src/components/HeroDashboard.tsx`** (новый) — стилизованный мок интерфейса/дашборда справа (карточка проекта + мини-график + бейджи «AI / MVP / Landing»), с органическим blur-объектом на фоне.
- **`src/styles.css`** — добавляю семантические токены для палитры (`--hero-bg`, `--hero-cream`, `--accent-green`, `--accent-coral`, градиенты и тени), подключаю шрифты через `<link>` в `__root.tsx` head.
- **`src/routes/__root.tsx`** — только добавляю `<link>` для Google Fonts (Instrument Serif + Inter). Заголовок/мету не трогаю (пользователь сказал «остальное не трогай»).

### CTA-кнопки
- «Посмотреть проекты» — залитая зелёным (`--accent-green`), ведёт на `#projects`.
- «Связаться» — залитая коралловым (`--accent-coral`), ведёт на `#contact`.
- Обе с мягкой тенью и hover-подъёмом.

### Адаптив
- Desktop ≥ `lg`: две колонки (текст 55% / дашборд 45%).
- Tablet `md`: две колонки, уменьшенный дашборд.
- Mobile `< md`: одна колонка, дашборд под текстом, H1 уменьшается (clamp), кнопки становятся полноширинными и располагаются вертикально с сохранением цветовых акцентов; SVG-волна масштабируется по ширине.
- Использую `grid-cols-[minmax(0,1fr)_auto]` + `min-w-0` + `truncate` по гайду responsive-layout.

### Что НЕ трогаю
- Существующую мету/SEO, favicon, роутинг, error/notFound компоненты, tokens shadcn — оставляю как есть, только **добавляю** новые токены рядом.
